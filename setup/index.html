■ インストール手順の流れ

前程： wordpress 4.9.8 がセットアップされている事。
=> 上記version 以外は、動作しない場合があります。

  • 拡張分の、テーブル追加

  • インストール

  • config設定

■ github から、拡張部分のコードを取得。

git clone 又は、ダウンロード

https://github.com/kuc-arc-f/match-publish-simply

=> 設置例は、wordpress を。ROOT/wpmatch に
インストールした場合です。

=> ある程度、LAMP系の設定知識がある方を対象にしています。

=> ＤＢ接続先等、phpシステム全体の設定
